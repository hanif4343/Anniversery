## ⚠️ প্রথমেই এই ধাপটা করতে হবে (গুরুত্বপূর্ণ)

আমি যে ফাইলগুলো দিয়েছি তাতে শুধু `lib/`, `pubspec.yaml` আর ওয়ার্কফ্লো আছে — কিন্তু একটা আসল Flutter প্রজেক্টে `android/` এবং `ios/` নামের প্ল্যাটফর্ম ফোল্ডারও লাগে (এগুলো অনেক বড়, তাই আমি সেগুলো নিজে বানাইনি)। এগুলো বানাতে আপনার কম্পিউটারে **Flutter SDK ইনস্টল থাকা লাগবে** (একবারই, শুধু এই ধাপটার জন্য):

1. Flutter ইনস্টল না থাকলে flutter.dev থেকে ইনস্টল করুন (Windows/Mac/Linux — একবারের কাজ)
2. এই zip আনজিপ করা ফোল্ডারে গিয়ে টার্মিনালে লিখুন:
```bash
flutter create .
```
এটা existing `lib/` ফাইল **মুছবে না**, শুধু `android/`, `ios/` ইত্যাদি মিসিং ফোল্ডার বানিয়ে দেবে।
3. তারপর `flutter pub get` চালান, দেখুন এরর আসে কিনা

**যদি আপনার কম্পিউটারে Flutter ইনস্টল করতে সমস্যা হয়** — আমাকে জানান, আমি বিকল্প হিসেবে GitHub Codespaces (ব্রাউজারেই টার্মিনাল, ইনস্টল লাগে না) দিয়ে কীভাবে করবেন সেটার ধাপে ধাপে গাইড দিয়ে দেব।

## GitHub এ তোলার ধাপ (প্রথমবার)

1. GitHub এ একটা **Private** রিপো বানান (নাম যা খুশি, যেমন `love-journey-app`)
2. এই পুরো ফোল্ডারটা ডাউনলোড করে আনজিপ করুন (উপরের ধাপ অনুযায়ী `flutter create .` চালানোর পর)
3. টার্মিনাল/গিট বাশ খুলে ফোল্ডারে যান, তারপর:

```bash
git init
git add .
git commit -m "Day 1: project base + GitHub Actions build"
git branch -M main
git remote add origin https://github.com/আপনার-ইউজারনেম/love-journey-app.git
git push -u origin main
```

4. GitHub এ রিপোর **Actions** ট্যাবে যান — দেখবেন "Build APK" workflow অটোমেটিক চলছে
5. ৫-১০ মিনিট পর ওই workflow run এর নিচে **Artifacts** সেকশনে `love-journey-apk` পাবেন — ওটাই ডাউনলোড করে ফোনে ইনস্টল করুন

নতুন কোড ফাইল যোগ/পরিবর্তন হলে শুধু:

## Personal Access Token লাগবে যদি push এ পাসওয়ার্ড চায়

GitHub এখন সরাসরি পাসওয়ার্ড দিয়ে push নেয় না। প্রথমবার push করার সময় যদি আটকায়:
1. GitHub → Settings → Developer settings → Personal access tokens → Generate new token (classic)
2. `repo` স্কোপ টিক দিন, Generate করুন
3. push করার সময় পাসওয়ার্ড এর জায়গায় এই টোকেন পেস্ট করুন

## Day 2 এর পর — Android পারমিশন যোগ করা (জরুরি)

ছবি/ভিডিও বাছাই, মিউজিক ফাইল বাছাই, আর ভয়েস রেকর্ড করতে হলে অ্যাপের কিছু পারমিশন লাগবে। `flutter create .` চালানোর পর তৈরি হওয়া এই ফাইলটা খুলুন:

```
android/app/src/main/AndroidManifest.xml
```

`<manifest ...>` ট্যাগের ভিতরে, `<application ...>` ট্যাগের ঠিক আগে এই লাইনগুলো যোগ করুন:

```xml
<uses-permission android:name="android.permission.RECORD_AUDIO" />
<uses-permission android:name="android.permission.READ_MEDIA_IMAGES" />
<uses-permission android:name="android.permission.READ_MEDIA_VIDEO" />
<uses-permission android:name="android.permission.READ_MEDIA_AUDIO" />
<uses-permission android:name="android.permission.READ_EXTERNAL_STORAGE" />
```

এটা না করলে ছবি/ভিডিও/মিউজিক বাছাই বা ভয়েস রেকর্ড কাজ করবে না (অ্যাপ ক্র্যাশ করতে পারে বা পারমিশন চাইবে না)।

যোগ করার পর যথারীতি:
```bash
git add .
git commit -m "Day 2: moment editor + media permissions"
git push
```

## Day 2 এ যা যোগ হলো

- প্রতিটা Chapter এ ট্যাপ করলে সেই chapter এর মোমেন্ট লিস্ট দেখা যায়
- "নতুন মোমেন্ট" বাটনে পূর্ণাঙ্গ ফর্ম: টাইটেল, তারিখ, বর্ণনা
- একাধিক ছবি আপলোড (গ্যালারি থেকে), ছবি রিমুভ করা যায়
- ভিডিও আপলোড (একটা)
- মিউজিক ফাইল আপলোড (mp3 ইত্যাদি)
- নিজের কণ্ঠে ভয়েস রেকর্ড করা (mic বাটন চেপে শুরু/শেষ)
- এন্ট্রি ইফেক্ট বাছাই (Day 4 তে এগুলো আসলেই অ্যানিমেট হবে)
- ছবির লেআউট বাছাই (single/collage/polaroid/stack)
- মোমেন্ট লিস্টে drag করে reorder করা যায়, swipe করে delete করা যায়
- সব ছবি/ভিডিও/অডিও ফোনের নিজস্ব স্থায়ী ফোল্ডারে কপি হয়ে সেভ থাকে (গ্যালারি ক্যাশ মুছে গেলেও হারাবে না)

## Day 3 এ যা যোগ হলো

- "Begin Journey" বাটনে এখন আসল Story Intro স্ক্রিন খোলে (লাইন বাই লাইন টেক্সট ফেড-ইন)
- এরপর Story Player — সব chapter এর সব মোমেন্ট একে একে সোয়াইপ করে দেখা যায় (বাম-ডানে সোয়াইপ)
- উপরে প্রগ্রেস বার আর chapter এর নাম দেখা যায়
- প্রতিটা দৃশ্যে ছবির লেআউট (single/collage/polaroid/stack) অনুযায়ী দেখায়, ভিডিও থাকলে ভিডিও প্লেয়ার দেখায় (ট্যাপ করে play/pause)
- প্রতিটা দৃশ্য হালকা fade + slide অ্যানিমেশন দিয়ে আসে
- Chapter অনুযায়ী আলাদা ব্যাকগ্রাউন্ড রঙ (এখনো গ্রেডিয়েন্ট কালার — Day 4 তে আসল ব্যাকগ্রাউন্ড ইমেজ/পার্টিকল যোগ হবে)

**টেস্ট করবেন কীভাবে:** Day 2 এর ফর্ম দিয়ে অন্তত ২-৩ টা chapter এ মোমেন্ট যোগ করে তারপর Home থেকে "Begin Journey" চাপুন — পুরো ফ্লো দেখতে পাবেন।

## Day 4 এ যা যোগ হলো

Creator Mode এর ফর্মে যে ইফেক্ট বাছাই করবেন সেটা এখন **সত্যিই প্লে হয়** Story Mode এ:

- 💌 খাম খুলে চিঠি বের হওয়া (envelope_letter)
- 💍 আংটি স্লাইড করে এসে বসা (ring_slide)
- 🌹 গোলাপ পাপড়ি ঝরা (rose_petal_fall)
- ❤️ হার্ট ভেসে ওঠা (heart_float)
- 📖 বইয়ের পাতা উল্টানো — 3D flip (page_turn)
- 📸 ক্যামেরা ফ্ল্যাশ (camera_flash)
- ✨ Golden Dust ঝরা (golden_dust)
- 🎆 আতশবাজি বার্স্ট (fireworks) — Anniversary/ending দৃশ্যের জন্য
- সাধারণ Fade আর Zoom-in ও আছে

**সাজেশন — কোন মোমেন্টে কোন ইফেক্ট দেবেন:**
- প্রপোজাল মুহূর্ত → `envelope_letter` অথবা `ring_slide`
- বিয়ের ছবি/ভিডিও → `camera_flash` বা `fade`
- Chapter শুরুর প্রথম মোমেন্ট → `page_turn`
- রোমান্টিক/emotional মুহূর্ত → `rose_petal_fall` বা `heart_float`
- শেষ Anniversary দৃশ্য → `fireworks`
- বাকি সাধারণ মুহূর্তগুলো → `fade` (ওভারইউজ করলে ইফেক্ট আর স্পেশাল লাগবে না)

## Day 5 এ যা যোগ হলো

- প্রতিটা Chapter এ ট্যাপ করলে অ্যাপবারে একটা 🎵 আইকন দেখবেন — সেখান থেকে সেই chapter এর ব্যাকগ্রাউন্ড মিউজিক (mp3) সেট করতে পারবেন
- Story Mode এ ওই chapter এর যেকোনো মোমেন্টে থাকলে মিউজিক লুপ হয়ে বাজবে; পরের chapter এ গেলে অটোমেটিক পরিবর্তন হবে
- কোনো নির্দিষ্ট মোমেন্টে যদি Day 2 এর ফর্মে আলাদা মিউজিক দিয়ে থাকেন, সেটা chapter এর ডিফল্টের বদলে বাজবে
- Voice note রেকর্ড করা থাকলে Story Mode এ "আমার কণ্ঠে শুনুন" বাটন দেখাবে — চাপলে ব্যাকগ্রাউন্ড মিউজিক আপনা-আপনি নিচু হয়ে ভয়েস বাজবে
- উপরে ডানদিকে mute/unmute বাটন যোগ হয়েছে

**✅ Core সম্পূর্ণ!** — এখন থেকে অ্যাপটা শুরু থেকে শেষ পর্যন্ত সম্পূর্ণ, ইনস্টলযোগ্য এবং দেখানোর মতো। বাকি দিনগুলো (৬-৭) পলিশ, আসল ছবি বসানো আর টেস্টিং এর জন্য।

## Day 6 এ যা যোগ হলো (পলিশ ও টেস্টিং টুলস)

- **ছবি/ভিডিও মিসিং হলে ক্র্যাশ করবে না** — কোনো ফাইল ডিলিট/মুভ হয়ে গেলে এখন একটা placeholder icon দেখাবে, অ্যাপ বন্ধ হবে না
- **Chapter Preview বাটন** — Creator Mode এ প্রতিটা chapter এর ভিতরে ▶️ আইকনে চাপলে শুধু ওই chapter টাই Story Mode এ চালিয়ে দেখা যায় (পুরো জার্নি প্রথম থেকে না দেখেই টেস্ট করা যাবে — এতে টেস্টিং অনেক দ্রুত হবে)
- **Full Preview বাটন** — Creator Mode এর হোম স্ক্রিনের উপরে ▶️ আইকনে চাপলে পুরো Story Mode শুরু থেকে দেখা যায়
- **প্রগ্রেস সামারি** — Creator Mode হোম স্ক্রিনের উপরে এখন দেখাবে মোট কতগুলো মোমেন্ট আছে আর কয়টা chapter এ কনটেন্ট বাকি ("এখনো খালি" কমলা রঙে হাইলাইট থাকবে)

**এখন Day 6 এ আপনার আসল কাজ:**
1. প্রতিটা chapter এ ঢুকে আসল ছবি/ভিডিও বসান (Creator Mode থেকে)
2. যেসব মুহূর্তের ছবি নেই, সেগুলোর জন্য AI দিয়ে বানানো ইলাস্ট্রেশন আপলোড করুন, আর "AI-generated" চেকবক্সে টিক দিন
3. প্রতিটা chapter শেষ হলে সেই chapter এর ▶️ Preview বাটনে চেপে দেখে নিন কেমন লাগছে
4. সব chapter শেষ হলে হোম স্ক্রিনের উপরের ▶️ বাটনে চেপে শুরু থেকে শেষ পর্যন্ত একবার পুরোটা দেখুন

## অ্যাপের নাম/আইকন সুন্দর করা (অপশনাল, ৫ মিনিটের কাজ)

ডিফল্টভাবে ফোনে অ্যাপটা "love_journey" নামে দেখাবে। সুন্দর নাম দিতে চাইলে:

**নাম পরিবর্তন:** `android/app/src/main/AndroidManifest.xml` ফাইলে `android:label="love_journey"` খুঁজে বের করে বদলে দিন, যেমন:

```xml
android:label="Our Love Journey"
```

**আইকন পরিবর্তন (অপশনাল, একটু বেশি কাজ):** একটা হার্ট/কাপল থিমের ছবি (1024x1024px) জোগাড় করে `flutter_launcher_icons` প্যাকেজ ব্যবহার করে বসাতে পারেন — এটা না করলেও অ্যাপ পুরোপুরি কাজ করবে, শুধু ডিফল্ট Flutter আইকন দেখাবে। চাইলে বলবেন, এই ধাপটার বিস্তারিত গাইড আলাদাভাবে দিয়ে দেব।

## ফোল্ডার স্ট্রাকচার
```
lib/
  models/       -> Chapter, Moment ডেটা মডেল
  services/     -> StorageService (ডাটাবেস), MediaService (ছবি/ভিডিও/মিউজিক/ভয়েস),
                    AudioController (background music + voice note প্লেব্যাক)
  widgets/      -> PhotoLayout, VideoMomentPlayer, background gradients
  widgets/effects/ -> সব entrance effect (envelope, ring, petals, fireworks ইত্যাদি)
  screens/      -> সব UI স্ক্রিন (home, creator lock/home, chapter moments,
                    moment editor, story intro, story player)
  main.dart     -> এন্ট্রি পয়েন্ট
assets/
  music/        -> এখানে আপনার mp3 ফাইল রাখবেন (Day 5)
  backgrounds/  -> ব্যাকগ্রাউন্ড ইমেজ (Day 4)
  animations/   -> Lottie JSON অ্যানিমেশন ফাইল (Day 4)
.github/workflows/build.yml -> অটো বিল্ড পাইপলাইন
```
